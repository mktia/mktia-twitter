各種機能

片思いチェック

片思いしているアカウントのリスト kataomoi を作成します.


片思われチェック

片思いされているアカウントのリスト notfollow を作成します.

※鍵アカウントはチェックの対象外です.




使用上の注意

  1. 一度にリストに追加できるアカウントの数は API 制限により 180 が限界ですが、安全のため 150 に設定しています.
  2. 以前作成されたリストが存在すると正常に動作しないことがあります.



禁則事項

一日に何度も実行しないでください.

このプログラムは beta 版です.

事前にテストしていますが、何が起こるかわかりません.

実行回数によっては API を使いすぎて、アカウントが凍結する可能性もあります.
