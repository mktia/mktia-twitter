片思いチェック

片思いしているアカウントのリスト kataomoi を作成します。


片思われチェック

片思いされているアカウントのリスト notfollow を作成します。

※鍵アカウントはチェックの対象外です。


各機能を利用すると、処理結果を自動でツイートします。

使用上の注意

  1. 終了画面が出る前に閉じると、プログラムは強制終了します。

  2. 一度にリストに追加できるアカウントの数は API 制限により 180 が限界ですが、安全のため 150 に設定しています。

  3. 以前作成されたリストが存在すると正常に動作しません。

禁止事項

一度実行したら、連続して実行しないでください。

API を使いすぎて、アカウントが凍結する可能性があります.
